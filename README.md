# MLPT - Math Learning Python Toolkit

MLPT is a small command-line Python project that groups several mathematical tools into a single program.

The goal of this project is to practice Python programming fundamentals and algorithmic thinking.

## Features

- Mini calculator
- Prime factor decomposition
- Factorial calculator
- GCD and LCM calculator
- Geometry calculations (area, perimeter, and volume)

## Requirements

Python 3.x

## How to Run

Clone the repository and run:

python main.py
